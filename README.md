# Commands

Task followed by any of the following cmds

| Command |            Desc             |
| :------ | :-------------------------: |
| add     |         add a task          |
| check   |   mark a task as complete   |
| done    | delete a task from the list |
| clear   |       clear all tasks       |
| display |      display all tasks      |
